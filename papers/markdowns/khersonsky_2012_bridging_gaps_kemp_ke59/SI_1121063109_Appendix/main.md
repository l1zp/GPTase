# B. Kinetic analysis of KE59 variants

Enzymatic characterization. For the kinetic characterization, the reactions were started by adding $1 5 0 ~ \mu \mathrm { l }$ of substrate (final concentration - $0 . 1 3 \mathrm { - } 1 . 0 5 ~ \mathrm { m M }$ , or $0 . 0 6 5 { \cdot } 0 . 5 2 5 \mathrm { m M }$ , in $2 5 ~ \mathrm { m M }$ HEPES $\mathrm { p H } ~ 7 . 2 5$ with $1 0 0 \ \mathrm { m M \ N a C l } )$ , to $5 0 \mu \mathrm { l }$ of enzyme in storage buffer, as previously described (2, 3).

$p H \cdot$ -rate profile. $\mathbf { k } _ { \mathrm { c a t } }$ and $\mathrm { K } _ { \mathrm { m } }$ values were determined with 5-nitrobenzisoxazole at $\mathrm { p H } ~ 5 \textmd { - }$ 8.5 for KE59 variants R2-4/3D, R4-5/11B, R8-2/7A, R9-1/4A, and R13-3/11H (2, 3). Initial velocities $\left( \mathbf { v } _ { 0 } \right)$ were determined at eight different substrate concentrations (0.13- $1 . 0 5 ~ \mathrm { m M }$ or $0 . 0 6 5 { \cdot } 0 . 5 2 5 \mathrm { m M } )$ . The buffers used (at $5 0 ~ \mathrm { m M }$ ) were: Citrate (pH 5.0-5.5), MES $\mathrm { ( p H ~ } 5 . 5 – 6 . 5 )$ , and Bis-tris propane $\mathrm { ( p H ~ 6 . 5  – 8 . 0 ) }$ at $5 0 ~ \mathrm { m M }$ . At $\mathrm { p H } 5 . 5$ and 6.5, the activity was measured with both relevant buffers. The ionic strength was adjusted to a total of 0.1M with NaCl.

Data analysis. Kinetic parameters $( k _ { c a t } , K _ { M } , k _ { c a t } / K _ { M } )$ were obtained by fitting the data to the Michaelis-Menten equation $[ \mathrm { v } _ { 0 } { = } k _ { c a t } [ \mathrm { E } ] _ { 0 } [ \mathrm { S } ] _ { 0 } / ( [ \mathrm { S } ] _ { 0 } { + } K _ { M } ) ] .$ , using the program Kaleidagraph 5.0. All the data presented are the averages of at least three independent experiments, with the standard deviations between the repeated experiments typically being $< 1 0 \%$ .

# C. Structural analysis of KE59 variants

# Protein crystallization, data collection, and refinement.

KE59 variants subjected to crystallization were purified by Ni-NTA HiTrap chelating HP column (GE), followed by gel filtration (HiLoad 16/60 Superdex $7 5 ^ { \mathrm { T M } }$ , GE). Crystals of KE59 variants were obtained by the sitting-drop, and the hanging-drop vapor diffusion and microbtach under oil methods using the Mosquito robot (TTP LabTech, Inc., Cambridge, MA, USA). The protein concentration used for crystallization of all variants was $2 0 { - } 3 5 ~ \mathrm { m g / m l }$ and all the crystals were grown at $4 ~ ^ { \circ } \mathrm { C }$ . Diffraction data were integrated, scaled, and reduced using the HKL2000 program package (4). The structure of R8-2/7A apo KE59 variant was solved by molecular replacement with the program PHASER (5), using the refined structure of indoleglycerolphosphate synthase from Sulfolobus Solfataricus (PDB entry 1IGS) as a model. The structures of the remaining KE59 variants were solved using the R8-2/7A apo KE59 variant structure as a model. All the steps of atomic refinement were carried out with the program CCP4/Refmac5 (6). The models were built to !a-weighted, $2 \mathrm { F _ { o b s } }$ $- \operatorname { F } _ { \mathrm { c a l c } }$ , and $\mathrm { F _ { o b s } - F _ { c a l c } }$ maps using the program COOT (7). Details of the data collection and structure refinement are described in Table S7. The KE59 models were evaluated with the program PROCHECK!"#.

# D. Molecular Dynamics analysis of KE59 variants

Materials and Methods. Molecular Dynamics simulations were carried out on the wild type KE59 design (KE59wt), and the evolved variants R1-7/10H, R2-4/3D, R4-6/6A, R5- 11/5F, R8-2/7A, R9-1/4A, and R13-3/11H. Each variant was prepared for nine independent MD simulations: one apo-MD, and two MDs for each of the substrates 5-nitro-6-chlorobenzisoxazole, 5-nitro-benzisoxazole, 5,7-dichloro-benzisoxazole, and 4,6-dichlorobenzisoxazole. These were initiated with the substrates in two distinct orientations – one with the heterocycle pointing towards the bottom of the active site (as anticipated from the computational design) and one with the heterocycle pointing towards bulk solvent (corresponding to a 180 degree rotation). Substrate parameters were generated within the antechamber module of AMBER 10 (9) using the general AMBER force field, with partial charges set to fit the electrostatic potential generated at $\mathrm { H F } / 6 { - } 3 1 \mathrm { G } ^ { * }$ by RESP (10). The charges were calculated according to the Merz-Singh-Kollman scheme (11, 12) using Gaussian 03 (13). The structures immersed in a truncated octahedral box with a 10 Å buffer of TIP3P(14) water molecules. The systems were neutralized by addition of explicit counter ions. All simulations were carried out using the Stony Brook modification of the Amber 99 force field (15). A two stage geometry optimization approach was utilized, initially minimizing the positions of water molecules and ions, followed by an unrestrained minimization of all atoms. The systems were heated gently in six 50K, 50ps steps from 0 to 300K at constant volume periodic boundary conditions. Harmonic restraints of $3 0 \mathrm { k c a l / m o l }$ were applied to the solute, and the Langevin equilibration scheme was used to control and equalize the temperature. The time step was kept at 1 fs during the heating stages, allowing potential inhomogeneities to self-adjust. Each system was then equilibrated for 2 ns with a 2 fs timestep at a constant pressure of 1 atm. Water molecules were triangulated with the SHAKE algorithm such that the angle between the hydrogen atoms is kept fixed. A 20 ns production MD simulation was performed for each of the systems using PMEMD (16). Geometries and velocities were saved every 100 steps (0.2 ps) which resulted in a total of 100,000 frames from each production run. Long-range electrostatic effects were modeled using the particle-mesh-Ewald method (17).Post-MD data extraction and analysis were performed using the ptraj module of AMBER 10 (18)

Arrangement of E230, S210 and W109 in the active site. W109 and S210 are in direct competition for E230, and these are mutually exclusive contacts. The interaction between E230 and W109 has a dual effect: on one hand, it can fixate the positioning of E230 and draw its carboxylate oxygens away from S210, which coincides with increased $\mathbf { k } _ { \mathrm { c a t } }$ and $\mathrm { k _ { c a t } / K _ { m } }$ values (Fig. S13). On the other hand, the interaction between E230 and W109 can also weaken the basicity of E230. QM transition state calculations show that W109, when bound to one of the carboxylate oxygens of E230, can increase the reaction barrier by up to $1 . 5 \mathrm { k c a l / m o l }$ (Fig. S14).

For 6-chloro and 5-fluorobenzisoxazole, the initial linear stage of elimination was monitored. The kinetics of elimination at various buffer concentrations are represented at Fig. S19-S22. # F. Synthesis and characterization of substrates and inhibitors

Benzisoxazole, benzotriazole, and 6-chlorobenzotriazole were purchased from Sigma-Aldrich. 5-nitrobenzotriazole was purchased from Matrix Organics. 5-nitrobenzisoxazole was prepared by nitration of benzisoxazole (19). All the other benzisoxazoles were prepared from corresponding salicylaldehydes by the procedure of Kemp (20). $p K _ { a } ^ { L G }$ values of the salicylonitriles were obtained from the literature (Table S10).

# 5-nitro benzisoxazole (by procedure of Hollfelder (21)

1,2 benzisoxazole (1gr) was dissolved in $\mathrm { H } _ { 2 } \mathrm { S O } _ { 4 } ~ 9 8 \%$ (4ml) at $0 \%$ . A mixture of $9 8 \%$ $\mathrm { H } _ { 2 } \mathrm { S O } _ { 4 }$ $( 0 . 2 2 \mathrm { m l } )$ and $70 \%$ $\mathrm { H N O } _ { 3 }$ $( 0 . 5 2 \mathrm { m l } )$ ) was added slowly, the reaction was stirred 30 min at RT and then poured onto ice-water mixture $( 2 0 \mathrm { m l } )$ . The suspension was vacuumfiltered, and the precipitate was washed with ethanol, dried on vacuum and lyophilized. The DCM fractions were combined, dried with ${ \mathrm { N a } } _ { 2 } { \mathrm { S O } } _ { 4 }$ , filtered and evaporated.
